# StockEye - 股票之眼

一个现代化的股票盯盘前端应用，支持A股、港股、美股的实时监控。

## 功能特性

### 核心功能
- ✅ **股票搜索**: 支持根据股票代码或公司名称搜索
- ✅ **快速添加**: 搜索结果可一键添加到收藏列表
- ✅ **实时监控**: 显示股票当前价、涨跌幅、成交量等关键指标
- ✅ **市场支持**: 支持A股、港股、美股三大市场
- ✅ **本地存储**: 数据保存在浏览器，无需注册登录

### 数据展示
- ✅ **列表视图**: 展示股票代号、分类、当前价、涨跌幅、市值、交易状态
- ✅ **详情页面**: K线图、均线图（MA5/MA10/MA20）、详细技术指标
- ✅ **涨跌统计**: 5日、20日、120日涨跌幅分析

### 自动化功能
- ✅ **自动刷新**: 可配置自动刷新间隔（10秒/30秒/1分钟/5分钟）
- ✅ **并发请求**: 批量获取多只股票数据，响应快速
- ✅ **实时更新**: 显示最后更新时间

## 技术栈

- **框架**: Vue 3 + Vite
- **UI库**: Naive UI（轻量级现代化组件库）
- **图表**: ECharts（专业数据可视化）
- **状态管理**: Pinia
- **数据来源**:
  - A股: 新浪财经API、腾讯财经API
  - 港股/美股: 新浪财经API

## 使用说明

### 1. 添加股票
- 在顶部搜索框输入股票代码或公司名称
- 点击"搜索"按钮
- 从搜索结果中点击"添加"按钮

### 2. 查看列表
- 主页显示所有已添加的股票
- 实时显示价格、涨跌幅、市值等信息
- 点击"详情"查看更多数据
- 点击"删除"移除股票

### 3. 查看详情
- 点击股票列表中的"详情"按钮
- 查看K线图和均线图
- 查看5日、20日、120日涨跌幅
- 查看成交量、成交额等详细数据

### 4. 设置
- 点击右上角"设置"按钮
- 开启/关闭自动刷新
- 调整刷新间隔时间

## 注意事项

1. **API限制**: 使用的是免费公开API，稳定性不保证，建议不要过于频繁请求
2. **跨域问题**: 如遇到跨域问题，可能需要配置代理或使用浏览器插件
3. **数据延迟**: 免费API的数据可能有延迟，仅供参考
4. **浏览器兼容**: 建议使用Chrome、Firefox、Edge等现代浏览器
